Markdown语法

本文最后更新于：2023年1月18日 晚上

此篇用于记载Markdown语法😘

标题

1.使用=和-标记一级标题和二级标题

2.使用#标记1~6级标题

字体

删除线

文字添加删除线，在要添加的文本前后各加两个波浪线即可

eg：删除线

分隔线

可以使用三个或以上的“ * ”，“ - ”，“ _ ”来创立一条分隔线(行内不能有其他东西)

列表

1.无序列表

使用星号(*)、加号(+)或是减号(-)作为列表标记，这些标记后面要添加一个空格，然后再填写内容

eg:

  • 第一项
  • 第二项
  • 第三项

2.有序列表

使用数字并加上 . 号来表示

eg：

  1. 第一项
  2. 第二项
  3. 第三项

3.列表的嵌套

表嵌套只需在子列表中的选项前面添加两个或四个空格即可（缩进？）

eg：

  1. 第一项：
    • 第一项中的第一项
    • 第一项中的第二项
  2. 第二项：
    • 第二项中的第一项
    • 第二项中的第二项

贴代码

1.段落上的部分代码可用反引号（`）包起来

eg：

使用printf()函数输出Hello world

2.代码块

有两种方法可以创造代码块，我最常用的是使用 ( ``` ) 把代码包起来（另一种方法不太常用这里就不说啦）

链接

eg:

点击进入我的博客MoyichenのBlog

or

https://moyichen2007.github.io/

图片

可以使用Markdown语法插入图片，也可以使用<img>标签来进行图片的插入 ( 使用<img>标签时可以规定图片的宽度和高度，使用Markdown语法则无法规定 )

表格

eg：

表头 表头 表头
单元格 单元格 单元格
单元格 单元格 单元格
